# TaffyOnYoutube

这是一个 Chrome Manifest V3 浏览器扩展，会在 YouTube 和 Bilibili 网页版的视频封面缩略图上随机叠加“永雏塔菲 / Taffy”透明图片。扩展不会替换原始封面，只是在封面上添加一个可爱的塔菲贴纸层，适合做成整活、梗图风格的视频浏览体验。

## 功能

- 支持 YouTube：主页、搜索结果、视频页右侧推荐、订阅页等常见缩略图位置。
- 支持 Bilibili 网页版：主页、搜索结果、视频播放页推荐、空间页等常见封面位置。
- 随机选择多张塔菲透明素材。
- 使用洗牌后的页面顺序轮换素材，减少同一排连续出现同一只塔菲的情况。
- 随机左下角或右下角显示，并带有轻微大小和旋转变化。
- 同一个封面最多出现两只塔菲；如果出现两只，会左右分开，且不会使用同一张素材。
- YouTube Shorts 竖版封面会自动放大塔菲，并且固定只显示一只，普通横版封面大小保持不变。
- 使用 `MutationObserver` 和定时兜底扫描处理动态加载内容。
- 弹窗支持启用/禁用、调整大小、选择位置。

## 安装

1. 打开 `chrome://extensions`
2. 开启右上角的“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择这个项目文件夹
5. 打开或刷新 YouTube / Bilibili 页面

## 素材

塔菲透明图片放在 `images/processed/` 目录中，素材列表由 `images/assets.json` 管理。

如果添加新图片，请把处理好的透明 PNG 放进 `images/processed/`，并把相对路径加入 `images/assets.json`。
